# Installation Guide for coils2vmec

## Quick Start (New Machine)

Execute the following commands in order:

```bash
# 1. Clone repository (if needed)
git clone https://github.com/cdfantasy/coils2vmec.git
cd coils2vmec

# 2. Install in editable mode (registers package)
pip install -e .

# 3. Compile Fortran extension
make clean && make

# 4. Verify installation
python -c "from coils2vmec.fieldline_tracer import fieldline_tracer; fieldline_tracer.set_verbose(True); print('✓ Installation successful!')"
```

## Important Notes

- **Order matters**: Run `pip install -e .` BEFORE `make`
- The `.so` file must be in the project root directory
- After any Fortran code changes, run `make clean && make`
- Python code changes take effect immediately (no rebuild needed)

## Troubleshooting

### Error: "No module named '_fieldline_tracer'"

**Cause**: Missing compiled Fortran extension

**Solution**:
```bash
make clean && make
```

### Error: Import conflicts or outdated code

**Cause**: Duplicate `./coils2vmec/` directory

**Solution**: The makefile now auto-cleans this, but if it persists:
```bash
rm -rf ./coils2vmec/
make clean && make
```

## Directory Structure

```
coils2vmec/
  ├── src/coils2vmec/           # Python source code (editable)
  │   ├── fortran/              # Fortran source files
  │   └── *.py                  # Python modules
  ├── _fieldline_tracer.so      # Compiled extension (generated by make)
  └── makefile                  # Build system
```

## Dependencies

- Python 3.8+
- gfortran
- numpy
- f90wrap
- f2py (comes with numpy)
